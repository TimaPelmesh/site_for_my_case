Возможности

Всё для управления сетью

Восемь модулей, объединённых в единый интерфейс: от сбора конфигураций до AI-анализа и Telegram-уведомлений.

Дашборд
Статистика устройств и бэкапов в реальном времени, последние оповещения и графики активности через WebSocket.
Конфигурации
Просмотр, резервное копирование и поиск конфигураций. Diff-viewer с подсветкой различий между версиями.
Compliance
Автоматические проверки безопасности по правилам для каждого типа устройства. Отчёты и рекомендации по исправлению.
AI Анализ
Интеграция с LM Studio (локальный AI). Анализ безопасности и оптимизации конфигураций, экспорт в PDF. Кэш 24 часа.
Золотые стандарты
Создание эталонных шаблонов конфигураций с привязкой к типу устройства. Проверка соответствия одним кликом.
Telegram
Уведомления о критических событиях и изменениях. Mini App внутри Telegram с автоматической авторизацией.
Планировщик
Задачи с приоритетами и отслеживанием статусов. Гибкая фильтрация, сортировка и управление из веб-интерфейса.
Oxidized
Интеграция по REST API. Автоматический ежедневный сбор конфигураций с сетевых устройств через SSH/Telnet.

Как всё работает

Пять этапов — от сбора конфигураций до оповещений.

1
Сбор — Oxidized по расписанию подключается к устройствам и сохраняет конфиги.
2
Синхронизация — CORAX получает список узлов и конфиги через Oxidized REST API.
3
Compliance — сравнение конфига с эталонным шаблоном, формирование отчёта.
4
AI-анализ — конфиг отправляется в LM Studio, возвращаются рекомендации безопасности.
5
Оповещения — при изменениях формируются алерты и Telegram-уведомления.

Установка

Быстрый старт

Четыре шага до работающего CORAX в вашей сети.

bash 
# Клонировать репозиторий
git clone https://github.com/your-username/corax.git
cd corax

# Проверить версию Python (3.11+)
python --version
bash 
# Windows
python -m venv venv
venv\Scripts\activate

# Linux / macOS
python3 -m venv venv
source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt
.env 
# Обязательные
JWT_SECRET_KEY=ваш-ключ-минимум-32-символа
ENVIRONMENT=development
CORS_ORIGINS=http://localhost:8010

# Telegram (опционально)
TELEGRAM_BOT_TOKEN=your-bot-token
NCM_WEBAPP_URL=https://your-domain.com

# AI / LM Studio (опционально)
LMSTUDIO_ENABLED=1
LMSTUDIO_BASE_URL=http://127.0.0.1:1234/v1
LMSTUDIO_MODEL=vikhr-llama3.1-8b-instruct

# Oxidized (опционально)
OXIDIZED_ENABLED=1
OXIDIZED_BASE_URL=http://127.0.0.1:8888
bash 
# Создать администратора
python scripts/create_admin.py

# Режим разработки
python -m uvicorn backend.main:app \
  --host 127.0.0.1 --port 8010 --reload

# Production (4 воркера)
python -m uvicorn backend.main:app \
  --host 0.0.0.0 --port 8010 --workers 4

# Открыть браузер → http://127.0.0.1:8010
# Документация → http://127.0.0.1:8010/docs
corax — bash
